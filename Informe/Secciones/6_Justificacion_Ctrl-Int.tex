\section{Requerimiento de Control Integral}

En la práctica se dio cuenta que el error permanente ($e_{ss}$) no era nulo. Esta caracterísitica se debe a que la ganancia $k_1$ no era exactamente unitaria. Este comportamiento exhibe que esta topología requiere de cierta pre-amplificación para obtener un $e_{ss}=0$. Sin embargo, como se estudio en clase, esta pre-amplificación no solo requiere de trimming (dado que esta afectada también por la dispersión paramétrica de los componentes que la implementan), sino que mantiene cierta sensibilidad a las perturbaciones (por ejemplo, si se introduce ruido en una variable de estado se fuerza a que el sistema no siga la entrada). 

Si en cambio se implementa un control integral, el sistema elimina el $e_{ss}$ independientemente de nuestra capacidad para generar una ganancia deseada. Junto a este comportamiento, se dispone del que las perturbaciones no afectan la capacidad del sistema para seguir la señal de entrada.

En cuanto al interrogante de la necesidad de un control integral, no se dispone de una respuesta concluyente sin una caracterización del sistema físico modelado. El error permanente exhibido puede o no ser crítico de acuerdo a la aplicación, en función de esta se ha evaluar si los costos de implementar un control integral valen los beneficios previamente indicados.
